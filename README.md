# 📊 HR Analytics Dashboard – Employee Attrition Analysis

## 📌 Project Overview

This project focuses on analyzing employee attrition patterns using **Power BI**. The dashboard provides insights into workforce trends, employee turnover, and factors that influence attrition such as department, overtime, salary levels, and job roles.

The goal of this project is to help HR teams identify potential causes of employee turnover and support data-driven decision making for improving employee retention.

---

## 🗂 Dataset

The dataset used for this project is the **IBM HR Analytics Employee Attrition & Performance dataset**.

It contains information about employees such as:

* Age
* Department
* Job Role
* Salary
* Overtime status
* Years at company
* Gender
* Attrition status

Total Records: **1470 Employees**

---

## 🛠 Tools & Technologies

* **Power BI** – Data visualization and dashboard development
* **Power Query** – Data transformation
* **DAX (Data Analysis Expressions)** – Calculated metrics and measures
* **Excel / CSV Dataset**

---

## 📈 Key Metrics

The dashboard highlights the following key performance indicators:

* **Total Employees**
* **Attrition Count**
* **Attrition Rate (%)**
* **Average Salary**

---

## 📊 Dashboard Insights

The dashboard provides several analytical views:

* Attrition Count by **Department**
* Attrition Count by **Overtime**
* Employee Distribution by **Salary Level**
* Total Employees by **Gender**
* Attrition Count by **Job Role**

These insights help identify patterns in employee turnover and highlight areas where HR intervention may be required.

---

## 🔍 Key Findings

Some insights derived from the analysis:

* The **Research & Development department** shows the highest attrition.
* Employees working **overtime** tend to leave the company more frequently.
* A large proportion of employees fall into the **lower salary range**, which may influence attrition trends.
